# Driver License Generator 🚗

Driver License (မောင်းနှင်ခွင့်ပြunlock) ကို အွန်လိုင်းမှာ ဖန်တီးဖြစ်ပေးသည့် web application ဖြစ်ပါတယ်။

## Features ✨

- 📋 Driver License အွန်လိုင်းဖန်တီးခြင်း
- 🖼️ License Photo Upload
- 📝 Personal Information Form
- 🎨 Beautiful UI Design
- 💾 License PDF Download

## Installation & Setup 🛠️

### အင်စတောလုပ်ခြင်း

1. **Repository Clone လုပ်ပါ**
```bash
git clone https://github.com/mgdno47-dotcom/git-clone-https-github.com-papaguycodes-driver-license-generator.git
cd git-clone-https-github.com-papaguycodes-driver-license-generator
```

2. **Dependencies Install လုပ်ပါ (Python)**
```bash
pip install -r requirements.txt
```

3. **Application Run လုပ်ပါ**
```bash
python app.py
```

4. **Web Browser မှာ ဖွင့်ပါ**
```
http://localhost:5000
```
